# Projet_licence_Fin-d-Etude-ALS
ALS est une plateforme web d’apprentissage adaptatif conçue pour personnaliser le contenu pédagogique et les évaluations selon la progression de l’apprenant. Le projet intègre un agent IA d’assistance pédagogique, une architecture SPL modulaire et une gestion Agile avec Jira et GitHub.
